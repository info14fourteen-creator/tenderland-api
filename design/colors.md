# Tenderland Color Palette

These colors are fixed as the shared palette for the website and applications.

| Token | Hex | Usage |
| --- | --- | --- |
| `brand.navy` | `#25274D` | Primary dark background, header, main brand surfaces |
| `brand.slate` | `#464866` | Secondary dark surfaces, panels, muted UI blocks |
| `brand.mist` | `#AAABB8` | Borders, dividers, disabled states, secondary text on dark surfaces |
| `brand.cyan` | `#2E9CCA` | Primary accent, links, focus states, call-to-action highlights |
| `brand.blue` | `#29648A` | Deep accent, charts, hover states, supporting brand areas |
| `neutral.white` | `#FFFFFF` | Light text, light backgrounds |
| `neutral.black` | `#000000` | Dark text, contrast fallback |

## CSS Variables

```css
:root {
  --color-brand-navy: #25274D;
  --color-brand-slate: #464866;
  --color-brand-mist: #AAABB8;
  --color-brand-cyan: #2E9CCA;
  --color-brand-blue: #29648A;
  --color-white: #FFFFFF;
  --color-black: #000000;
}
```
